나도코딩이라는 유튜버의 파이썬 강의를 듣고 다음에 프로젝트도 한번 해봐야겠다는 생각을 가지고 있었는데 이번에 음성인식 인공지능에 관련된 영상이 올라와 직접 따라해보았다.
파이썬은 다양한 분야에서 넓게 사용되고있고, 패키지들을 통해 미리 만들어놓은 구조물들을 쉽게 사용할수 있어 만들어진 구조물들을 잘 이용하라고 들었었는데
이번에 실습을 해보면서 "패키지만 몇 개 설치했는데 이런것도 가능해..?"라는 생각이 들었고, 응용해서 기능을 추가한다면 일상생활에서도 충분히 사용할수있는 프로그램을 만들겠다는 생각이 들었다.
누군가 미리 만들어놓은 패키지를 통해 구현했지만 이를 잘 활용하는것도 파이썬을 잘 사용하는 것이라고 생각한다.
아직 인공지능에 대해서 배우지는 않았지만 간단한 실습이라 재밌었고, 매우 유망한 분야라는 생각이 들었다.
앞으로도 다양한 프로젝트를 실습해볼 생각인데 프로그래밍은 배울수록 다양한 기능들이 많다고 느끼고, 프로그래밍의 중요성에 대해서도 깨닫게 되는것 같다.
